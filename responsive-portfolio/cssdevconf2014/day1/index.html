      2. Minimize requests and round trip times
        • Avoid redirects
        • Concatenate CSS and JS to reduce requests
        • Use image sprites, espeically if on every page, so one requests
      3. Leverage cache, caceh rules everything around me
        • minimize times you query
        • Use content delivery network CDN
      4. Reduce payload
        • Gzip
        • Minifying
        • Compress and Serve scaled images
